OUTLOOK ADD-IN MIN (GitHub Pages)
=================================

Fichiers inclus :
- manifest.xml (à sideloader dans Outlook)
- web/taskpane.html
- web/taskpane.js
- web/taskpane.css
- web/assets/icon-16.png, icon-32.png, icon-80.png

Utilisation :
1) Pousse ce dossier dans un repo GitHub, active GitHub Pages (Settings → Pages).
2) Remplace <USER> et <REPO> dans manifest.xml par ton pseudo et le nom du repo.
3) Vérifie https://<USER>.github.io/<REPO>/web/taskpane.html s'affiche bien.
4) Dans Outlook (classique) → Mes compléments → Ajouter à partir d'un fichier → choisis manifest.xml.
5) Ouvre un mail, clique "Ouvrir le panneau" → vois le bouton "Ne rien faire".
